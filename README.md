# pokebot-slack - pokemon go slack notification
Cloned from https://github.com/tejado.

## Installation

    pip install .
 
## Usage

    python pokeslack.py

## Requirements
 * Python 2 or 3
 * requests
 * protobuf (>=3)
 * gpsoauth
 * geopy (only for pokecli demo)
 * s2sphere (only for pokecli demo)
 * slackclient
